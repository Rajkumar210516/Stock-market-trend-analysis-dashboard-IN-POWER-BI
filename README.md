# Stock-market-trend-analysis-dashboard-IN-POWER-BI
This project analyzes year-long stock trends of TCS, Reliance, and Infosys using Power BI, supported by an advanced Excel dataset and a presentation explaining the visuals.
The dashboard is fully interactive with slicing, filtering, KPIs, trend charts, and market insights.
